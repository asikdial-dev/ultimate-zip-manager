# 🚀 Ultimate ZIP Manager — Unlimited Upload

Upload files of ANY SIZE to free hosting with 2MB limits.

## 💡 How It Works

Uses **chunked upload** technology:
- Splits files into 1MB chunks
- Uploads one by one
- Bypasses 2MB server limit
- Reassembles on server

## ✨ Features

- Upload unlimited size files
- Drag & drop interface
- Real-time progress tracking
- Extract ZIP files
- File browser
- Batch delete

## 🛠️ Installation

1. Upload `ultimate-zip-manager.php`
2. Access via browser
3. Upload your massive files!
